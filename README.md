# NBA-Fantasy-Analytics
A personal SQL analytics project built to help with NBA fantasy basketball drafting and player evaluation. Built using PostgreSQL, pgAdmin, and Python with real 2024–25 NBA game log data.

 Project Overview 

This project analyzes 16,500+ NBA game logs from the 2024–25 season to help fantasy basketball players make smarter draft decisions. It covers player averages, custom fantasy scoring, head-to-head comparisons, and matchup breakdowns by opponent.
